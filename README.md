# SpringBoot+thymeleaf 请假系统

#### 注意
带有数据内容的数据库在“lssues”的提问中，可以在“已完成”中进行查看。

#### 介绍
springboot+thymeleaf 的一个请假系统

#### 项目部署

1. 将项目导入到idea中
2. 修改数据库配置文件
3. 修改maven路径并刷新maven仓库

#### 数据库
点击上方导航栏的 lssues ，然后点击"已完成"进行数据库详情查看。
